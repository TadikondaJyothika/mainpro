HTML QUESTIONS

  1. What is HTML?

    • HTML stands for HyperText Markup Language.
    • It is the standard language used to create and design web pages.

  2. What is the difference between HTML and XHTML

      HTML

      • HyperText Markup Language.
      • Based on SGML (Standard Generalized Markup Language)

      XHTML

      • eXtensible HyperText Markup Language
      • Based on XML (eXtensible Markup Language)

  3. What is Tag?

    • Tags are written inside angle brackets < >.
    • Most tags come in pairs: an opening tag and a closing tag.
      • Opening tag: <p>
      • Closing tag: </p>

  4. What is Element?

    • An HTML element is made up of an opening tag, some content, and a closing tag.
    • Example: <p> This is a paragraph.</p>

  5. What is the difference between HTML4 and HTML5?

      6. HTML4

      • Released in 1997.
      • Long doctype declaration.
      • No semantic tags like <header> or <footer>.
      • Needs plugins (like Flash) for audio and video.

      HTML5

      • Finalized in 2014.
      • Short doctype: <!DOCTYPE html>.
      • Includes semantic tags like <header>, <footer>>, <article>, <section>.
      • Built-in support for <audio> and <video>.

  6. What is Lists?
    • A list in HTML is a structured way of displaying related items, either in a specific order or without order. It uses list tags to group items together, making them easier to read and understand.

    1. Ordered list
    2. Unordered List
    3. Description List
